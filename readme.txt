Google Maps 簡単挿入プラグイン - Caret Ez Google Maps

== Description ==

### 投稿・ページ編集画面にボタンを追加し、Google Mapsを記事内に簡単に挿入することができるようになります。

使い方は、いたってシンプル。
編集領域内で、住所等が記述されている部分を反転し、Google Mapsボタンを押すだけ。
{GoogleMaps}～～～{/GoogleMaps}で囲まれた部分をGoogle Mapsに変換し、表示いたします。

== Installation ==

### ■注意事項
* Google Mapsで使用可能な住所等を入力してください。

### ■プラグインの有効化
管理ページ＞プラグイン＞Caret Ez Google Maps を有効にしてください。

### ■設定
不要です。
